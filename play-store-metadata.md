### Full Description
PDF Toolkit is a comprehensive mobile application designed to handle all your PDF document needs. Whether you're a student, professional, or anyone who works with PDF files, this app provides essential tools to manage your documents efficiently.

**Key Features:**
• **Merge PDFs** - Combine multiple PDF files into a single document
• **Split PDFs** - Extract specific pages or split documents into multiple files
• **Compress PDFs** - Reduce file size while maintaining quality
• **Image to PDF** - Convert photos and images to PDF format
• **Extract Pages** - Pull out specific pages from any PDF document
• **Rotate Pages** - Fix document orientation with easy rotation tools
• **Security Features** - Add password protection to your sensitive documents
• **Metadata Viewer** - View and edit PDF properties and information

**Why Choose PDF Toolkit:**
✓ User-friendly interface designed for mobile devices

✓ Secure - all processing happens locally on your device
✓ No internet connection required for most features

✓ Regular updates with new features

Perfect for students managing research papers, professionals handling business documents, or anyone who needs reliable PDF tools on the go.
